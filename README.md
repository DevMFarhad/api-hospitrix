## Overview

**Hospitrix** is a Hospital Management System backend built with **Node.js, Express, and TypeScript**, using **Prisma ORM** for database management.

## Features
- TypeScript-based Express backend
- Prisma ORM integration
- Structured project with routes, controllers, and middleware
- CORS, JSON parsing, and optional security headers (Helmet)
